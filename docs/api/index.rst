Core Modules
------------

The timber-joints library is organized into the following modules:

* **joints** - Joint geometry classes (Tenon, Dovetail, Lap, etc.)
* **alignment** - Beam positioning and alignment utilities
* **frame** - TimberFrame assembly system
* **barn** - High-level barn frame builder
* **analysis** - FEA integration for structural analysis
* **export** - IFC export and beam schedule generation
* **utils** - Helper functions and utilities
