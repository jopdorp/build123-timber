# Timber Joints v2

Simplified timber joinery library for build123d - inserting joint parts only.

## Joint Types

- **Beam** - Simple rectangular beam
- **LapJoint** - Half-depth cut at beam end
- **Tenon** - Centered projection at beam end
- **ShoulderedTenon** - Tenon with bearing shoulder
- **DovetailInsert** - Tapered projection

All joints represent only the **inserting/positive** part (no mortises, housings, or receiving cuts).
